Front-End Engineer Cheatsheet

This is a current working list of all HTML tags that I have learned about through Codeacademy's Front-End Engineering career path. At this time, this is a list have completed roughly 14% of the course. I will continue to update this list as I progress through through the program with the intent to roughly show my progress as we go.

HTML Tag Name Function
<!--...--> Comments Performed by pressing Command and the forward slash, this allows you to comment out lines. Used to be able to provide descriptions of code. Self-Closing
<!doctype> Doctype Used to estalish what type of document your file is open opening, usually written as followed: Doctype html. Self-Closing
<a> Anchor Specific a anchor (Hyperlink) Use for link in internal/external web documents. Needs a closing element
<article> Article Defines an article, an article represents a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable. Needs a closing element.
<Body> Body House the main body of the work that will be displayed on the headpage. While the head will include back-end information that is not seen, the body provides seen elements. Needs a closing element
<h1> Heading 1 By default the highest level of heading available. requires a closing element
